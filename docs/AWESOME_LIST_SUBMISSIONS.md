## 提交步骤

用 GitHub Web UI 提 PR 最简单：
1. Fork 目标 repo
2. 编辑 README.md，在对应分类下添加条目
3. 提交 Pull Request，标题格式: `Add GT Office`
4. 描述中说明: 项目信息 + 为什么适合这个分类
