### Decisions

1. MCP 工具“出现在工具列表”仅表示 server 启动成功，不等价于 bridge 业务链路可用。
2. `gto_health`/`gto_dispatch_task` 依赖 GT Office runtime（`~/.gtoffice/mcp/runtime.json` + 可连接端口）；桌面端未运行时失败是预期行为。
3. `startup_timeout_sec` 仅用于冷启动缓冲；优先确保配置唯一、协议兼容与 runtime 可达性。
4. 安装器必须保证幂等：重复执行 `mcp:install` 不得产生重复配置段。
