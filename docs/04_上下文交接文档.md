## 已完成

- Git 管理界面增加两层前端补刷兜底：
  - `apps/desktop-web/src/features/git/GitPane.tsx` 中的 `GitOperationsPane` 挂载时自动执行一次 `refreshAll()`，确保用户点击进入 Git 管理界面时会主动拉一次最新状态，而不是完全依赖此前 watcher 事件是否送达。
  - 同一组件在切换 Git 筛选区 `all/staged/unstaged` 时，除首次初始化外会自动执行一次 `refreshSummary()`，保证切换“全部 / 已暂存 / 未暂存”时同步补刷变更文件列表。
  - `apps/desktop-web/src/features/git/useGitWorkspaceController.ts` 暴露 `refreshSummary`，供 Git 面板在不重拉 branch/stash/history 的情况下执行轻量 summary 同步。
