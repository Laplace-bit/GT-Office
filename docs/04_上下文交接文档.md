# vbCode 上下文交接文档

## 当前里程碑与本轮目标

- 里程碑：M2 MVP 完整
- 任务：T-175
- 本轮目标：把任务中心中“选择 agent + 发送任务”的核心能力封装成一个可全局唤起的任务派发浮层，并补齐快捷键设置。

## 已完成

- **全局任务派发浮层**：
  - `apps/desktop-web/src/features/task-center/GlobalTaskDispatchOverlay.tsx`：新增 portal 式全局浮层，支持任意界面唤起、Esc 关闭、`Mod+Enter` 发送、透明度即时调节。
  - `apps/desktop-web/src/features/task-center/GlobalTaskDispatchOverlay.scss`：新增桌面工作台风格浮层皮肤，保留当前界面上下文并强化 keyboard focus 可见性。
  - `apps/desktop-web/src/features/task-center/TaskCenterPane.tsx` / `.scss`：补齐 `overlay` 变体与发送快捷键提示，让现有任务中心编辑器可复用于侧栏和全局浮层两种容器。
- **快捷键与设置持久化**：
  - `apps/desktop-web/src/features/keybindings/shortcut-bindings.ts`：新增命令 `task.center.quick_dispatch`，默认键位 `Mod+Shift+K`，并补齐 keystroke 序列化、格式化展示、录制解析能力。
  - `apps/desktop-web/src/features/settings/TaskDispatchPreferences.tsx` / `.scss`：新增设置中心“快捷键”配置卡，当前仅管理任务派发快捷键，后续其他快捷键也应统一归入此页。
  - `apps/desktop-web/src/features/settings/SettingsModal.tsx`：新增独立 `Keybindings` 页签；浮层透明度不再进入设置中心，仅在浮层内调节。
  - `apps/desktop-web/src/shell/layout/ShellRoot.tsx`：接入全局快捷键、设置加载/持久化、录制态防误触锁与 overlay 渲染。
- **文档同步**：
  - 已更新 `docs/03`、`docs/05`、`docs/06`，补充本轮需求、验收与设置契约。
