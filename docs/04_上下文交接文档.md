# GT Office 上下文交接文档

## 补充更新（2026-03-27）

- 已修复 macOS 命令行卡片输入删除偶发视觉不同步问题。根因不是 PTY 或删除键映射错误，而是 `apps/desktop-web/src/features/terminal/StationXtermTerminal.tsx` 的 macOS WebKit deferred input fallback 在原生 input 分支外部清空了 xterm helper textarea，破坏 xterm 自己的 composition/textarea 同步，导致 Backspace 已发送到终端但 helper textarea 仍残留旧字形。
- 修复方式：将 deferred input 分支提炼为 `macos-webkit-ime-workaround.ts` 的纯决策函数，保留 xterm 对 helper textarea 的所有权，不再在 fallback 中手动 `textarea.value = ''`。
- 已补回归用例：`apps/desktop-web/tests/macos-webkit-ime-workaround.test.ts` 新增“forward/skip 时保留 helper textarea 状态”验证。
- 验证：`tsc -p apps/desktop-web/tsconfig.tests.json --outDir /tmp/gto-desktop-web-test-dist-green` + `node --test /tmp/gto-desktop-web-test-dist-green/tests/macos-webkit-ime-workaround.test.js`、`tsc -p apps/desktop-web/tsconfig.app.json --tsBuildInfoFile /tmp/gto-desktop-web-tsconfig.app.tsbuildinfo`、`tsc -p apps/desktop-web/tsconfig.node.json --tsBuildInfoFile /tmp/gto-desktop-web-tsconfig.node.tsbuildinfo`、`vite build --configLoader runner --outDir /tmp/gto-desktop-web-dist`。
