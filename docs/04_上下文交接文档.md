## 已完成

- **Git 管理界面视觉收敛**：
  - `apps/desktop-web/src/styles/features/git/_pane.scss`：将 `git-pane` 保留为唯一主容器，把原来每个 section 的独立卡片边框/圆角/背景改成分隔线式区块；header、branch info、count、error bar 改成更轻的工具条语言；列表、输入框、stash item、graph detail、graph actions 都压掉多余材质和 hover 缩放。
  - `apps/desktop-web/src/styles/features/git/_diff-viewer.scss`：diff viewer 改为与 history pane 共用同一主面板语义，移除额外卡片底；header、badge、mode toggle、hunk header、gutter 和 body 背景统一为表格式层次，而不是独立玻璃块。
- **结果**：
  - Git 操作页与历史页现在更接近“桌面管理台 + 表格/工具条”观感，层级从多层卡片叠加收敛成单容器内分带。
- **未触达内容**：
  - 本轮没有修改 git 交互逻辑、命令流、虚拟列表策略，也没有新增前端状态。
