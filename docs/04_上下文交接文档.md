## 已完成

- **外部事件已按两端关联键建模并分组**：
  - `apps/desktop-web/src/shell/layout/ShellRoot.tsx` 为 `ExternalChannelEventItem` 补齐 `traceId/accountId/peerId/targetAgentId/endpointKey/conversationKey`。
  - 新增“外部端点 key + target agent”会话键计算逻辑，并维护 `traceId -> context` 映射。
  - `onExternalRouted/onExternalDispatchProgress` 到达时会回填同 trace 事件的 target，从而把入站与出站事件对齐到同一会话分组。
- **通信通道面板已切到 Tab 分组视图**：
  - `apps/desktop-web/src/features/tool-adapter/CommunicationChannelsPane.tsx` 改为按 `conversationKey` 聚合并在顶部 tab 切换。
  - 顶部会话导航已从胶囊气泡改为扁平 tab，避免视觉“按钮气泡化”。
  - 删除“最近外部事件”标题，删除单条消息的时间/channel/status 元信息，消息区仅保留气泡正文（失败可附 detail）。
- **会话标签语义已收口到绑定关系**：
  - 会话 tab 标签改为 `channel bot 名称 - agent 名称`。
  - 文案生成优先使用 `ChannelRouteBinding.botName` 与 `targetAgentId -> station.name` 映射，不再展示长 `peer/account` 标识。
- **深色模式视觉已收口**：
  - `apps/desktop-web/src/features/tool-adapter/CommunicationChannelsPane.scss` 新增暗色主题变量覆盖，降低面板与气泡高亮反射，避免深色模式背景过亮。
