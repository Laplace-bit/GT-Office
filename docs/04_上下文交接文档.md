## 当前里程碑与本轮目标

- 里程碑：M2 MVP 完整
- 任务：T-171

## 已完成

- **前端配置中心交互重构**：
  - **模块化目录视图**：`AiProvidersSection.tsx` 现在只展示 Agent 卡片网格，移除冗余的内联工作区。
  - **交互式弹窗配置**：新增 `AiConfigOverlay` 通用组件，引入 `ClaudeConfigModal` 和 `LightAgentConfigModal` 承载复杂的配置逻辑。
  - **增强卡片设计**：`ProviderAgentCard.tsx` 新增“配置”按钮（已安装时可见）与“立即安装”按钮（未安装时可见），支持更清晰的状态反馈。
  - **审计 UI 移除**：按需删除了 `AuditLogView` 相关的所有前端组件与 SCSS 文件，保持界面纯净。
  - **UI/UX 深度打磨**：应用全量 Glassmorphism 材质、平滑的 Modal 弹出动画以及 Apple 风格的 Stepper 样式。
- **能力继承与安全**：
  - 完整继承了上一轮实现的 Claude 5步向导、API Key 托管、预览后应用以及 Codex/Gemini 的环境变量注入能力。
  - 保持后端 `vb-ai-config` 与 `vb-storage` 审计能力的完整性，仅在 UI 层进行 Progressive Disclosure。
- **代码清理**：
  - 彻底清理了已失效的内联 Workspace 组件、旧版引导卡片及其对应的 SCSS。
