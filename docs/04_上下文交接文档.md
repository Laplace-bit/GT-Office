## 已完成

- 后端 Git 刷新协调：
  - 新增 `apps/desktop-tauri/src-tauri/src/commands/git/status_coordinator.rs`，按 workspace 对 Git 状态刷新做去抖、singleflight 与指纹去重。
  - Git 命令完成后不再各自手动拼事件，而是统一走 coordinator 发 `git/updated`。
- watcher 与 Git 元数据联动：
  - `apps/desktop-tauri/src-tauri/src/filesystem_watcher.rs` 现在会在工作区文件变化和 `.git/HEAD`、`.git/index`、`refs/heads/*`、`packed-refs`、`MERGE_HEAD`、`rebase-*` 等关键元数据变化后调度 Git 状态刷新。
- 前端事件消费收口：
  - `apps/desktop-web/src/shell/layout/ShellRoot.tsx` 直接消费 `git/updated` 中的完整 summary 更新 Git 面板。
  - `apps/desktop-web/src/features/file-explorer/FileTreePane.tsx` 直接由 `git/updated.files` 构建文件树 Git 状态，不再在 `filesystem/changed` 后额外手动补刷 Git。
- 文档与测试：
  - 已更新 `docs/03`、`docs/05`、`docs/06` 与本交接快照。
  - 已补 watcher 相关测试用例。
