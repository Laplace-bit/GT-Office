# GT Office v0.1.7

GT Office `v0.1.7` adds GitHub Release based in-app update support for the desktop shell.

## Highlights

- Desktop settings now support checking for updates, viewing release notes, downloading and installing a newer signed release, and skipping a specific version.
- Tauri updater integration now reports status and download progress through dedicated settings commands and events.
- Release automation now generates signed updater artifacts and a merged `latest.json` manifest for GitHub-hosted update delivery.

## Reliability Notes

- Apple Silicon updater targets are normalized to `darwin-aarch64-app` so macOS ARM clients can match published update payloads.
- Startup auto-check notifications stay lightweight in the shell, while detailed progress state remains scoped to the settings UI.
- Missing release notes no longer break manifest generation; the release pipeline falls back to `Release <version>` when needed.
