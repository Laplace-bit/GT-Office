# GT Office v0.2.0

GT Office `v0.2.0` focuses on editor reliability, provider management fixes, and release-path stabilization.

## Highlights

- AI provider deletion now cleans up workspace-scoped configuration more reliably, reducing stale provider state after removal.
- Markdown preview handling was rebuilt around a dedicated renderer so local images and split-view preview behavior work correctly in the file editor.
- Desktop packaging and Rust CI fixes were merged so the Tauri release path includes the required bundled agent communication resources and passes the current build checks.
- Project README and release-facing documentation were refreshed for the new `v0.2.0` release line.

## Testing

- `npm run typecheck`
- `cargo check --workspace`
