# GT Office v0.1.9

GT Office `v0.1.9` enables signed in-app updates and refines the settings UI.

## Highlights

- Updater signing is now active: builds produce signed `.sig` artifacts and a merged `latest.json` manifest for GitHub Release delivery.
- The updater public key is baked into `tauri.conf.json` with a fallback in Rust, so the update feature works without environment variables during local development.
- Update preferences moved to the About tab for better visibility alongside version info.
- macOS window chrome and terminal CPR echo fixes from previous patches are included.

## Testing

- `npm run typecheck`
- `cargo test -p gtoffice-desktop-tauri update:: --lib`
- `npm --workspace apps/desktop-web run test:unit`
