# GT Office 文档索引

本目录用于沉淀 `GT Office` 项目的需求、架构、执行计划与协作上下文。

## 文档清单

1. `docs/01_需求与产品设计.md`
   - 产品目标、范围边界、功能需求、非功能指标、验收标准。
2. `docs/02_系统架构与模块目录设计.md`
   - 技术架构、模块职责、目录拆分、接口边界与数据流。
3. `docs/03_项目开发进度跟踪.md`
   - 里程碑计划、任务状态、风险阻塞、迭代节奏模板。
4. `docs/04_上下文交接文档.md`
   - Vibecoding 多窗口/跨会话交接标准，含 Next Agent Starter 无缝续接模板。
5. `docs/05_高质量功能设计_核心工作流.md`
   - Epic 拆分、用户故事、状态机、主/异常流程、验收用例。
6. `docs/06_API与事件契约草案.md`
   - Tauri Command/Event 契约、错误码、配置与存储 schema 草案。
7. `docs/07_依赖选型与精简清单.md`
   - 依赖白名单、禁止引入清单、按阶段选型与删除策略。

## 使用约定

- 需求变更先更新 `01` 与 `05`，再同步 `02` 与 `03`。
- 每次阶段性完成后更新 `03` 与 `04`。
- 团队协作规范统一以根目录 `AGENTS.md` 为准。
