## 🐛 已知问题和解决方案

### 问题1: 深色主题下输入框仍显示白色
**原因**: CSS变量未正确加载或有内联样式覆盖
**解决**: 检查是否有组件使用了内联`style`属性覆盖了背景色

### 问题2: 动效在某些浏览器不流畅
**原因**: 浏览器未启用硬件加速
**解决**: 
1. Chrome: 检查 `chrome://gpu/` 确认硬件加速已启用
2. Firefox: 在 `about:config` 中启用 `layers.acceleration.force-enabled`

### 问题3: WSL环境下样式未更新
**原因**: Vite热更新在WSL文件系统可能较慢
**解决**: 
1. 手动刷新浏览器 (Ctrl+Shift+R)
2. 或将项目移至WSL原生文件系统
